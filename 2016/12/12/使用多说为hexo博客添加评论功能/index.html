使用多说为hexo博客添加评论功能

发表于 2016-12-12 | 分类于 git ， hexo ， LeanCloud | | 阅读次数
nexT主题集成了多说，只需注册多说账号，注册多说账号：

1.创建多说,安装应用，即可创建一个应用；
2.填写域名，首页网址等，与github地址对应；
3.修改Hexo配置文件_config.yml,在末尾添加:

duoshuo_shortname: 多说名称


4.重新发布hexo即可成功；
~对于非nexT主题还要修改代码，nexT主题集成了多说评论。~

使用LeanCloud为nexT主题添加访问量：

1.登录LeanCloud,创建应用：



2.创建新的应用，应用名称可以随意取：




3.创建Class,在Class名称下点击创建Class,命名为Counter,选择无限制,用于记录浏览量:




4.返回到设置中，点击key，获取app_id和app_key,修改nexT主题下的_config.yml文件中的:



leancloud_visitors:
  enable: true
  app_id: 注册应用的app_id
  app_key: 注册应用的app_key


5.重新执行命令一下命令，完成设置：


hexo g
hexo d
